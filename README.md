# Egorozh.ColorPicker

[![NuGet version (Egorozh.ColorPicker.WPF)](https://github.com/egorozh/Egorozh.ColorPicker.WPF/blob/master/shield.svg)](https://www.nuget.org/packages/Egorozh.ColorPicker.WPF/)
